## Install

```sh
$ npm install --save-dev babel-preset-app-decorators
```

## Usage

### Via `.babelrc` (Recommended)

**.babelrc**

```json
{
  "presets": ["app-decorators"]
}
```

### Via CLI

```sh
$ babel script.js --presets app-decorators 
```

### Via Node API

```javascript
require("babel-core").transform("code", {
  presets: ["app-decorators"]
});
```
